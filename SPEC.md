# 项目交付材料收集系统 - 规格文档

## 1. 项目概述

- **项目名称**: 项目交付材料收集系统
- **功能**: 收集同事的交付材料，跟进项目进度
- **目标用户**: 项目经理、普通员工、管理员

## 2. 技术栈

- **前端**: Next.js 14 (App Router) + React + Tailwind CSS
- **后端**: Next.js API Routes
- **数据库**: SQLite (轻量，无需额外配置)
- **文件存储**: 本地文件系统 (`uploads/` 目录)
- **认证**: NextAuth.js (简单用户系统)

## 3. 数据模型

### User (用户)
```
- id: string
- username: string
- password: string (hashed)
- role: 'user' | 'admin'
- createdAt: datetime
```

### Project (项目)
```
- id: string
- projectName: string (项目名称)
- projectManager: string (项目经理)
- projectNumber: string (项目编号)
- userId: string (创建者)
- createdAt: datetime
- updatedAt: datetime
```

### Material (材料)
```
- id: string
- projectId: string
- materialName: string (材料名称)
- materialGroup: string (分组)
- images: string[] (图片路径数组，最多3张)
- createdAt: datetime
- updatedAt: datetime
```

## 4. 功能清单

### 4.1 认证系统
- [ ] 用户登录/登出
- [ ] 用户注册（可选，管理员可后台创建）
- [ ] 角色区分：普通用户、管理员

### 4.2 新建项目
- [ ] 输入项目名称（必填）
- [ ] 输入项目经理姓名（必填）
- [ ] 输入项目编号（必填）
- [ ] 提交后创建项目，跳转到项目详情页

### 4.3 项目详情/材料上传
- [ ] 显示项目信息（名称、经理、编号、创建时间）
- [ ] 按分组显示27项材料
- [ ] 每项材料：
  - 显示材料名称和备注
  - 上传按钮（最多3张）
  - 已上传图片可预览/删除
  - 可修改（重新上传替换）

### 4.4 历史项目
- [ ] 当前用户创建的所有项目列表
- [ ] 显示项目名称、经理、编号、创建时间
- [ ] 可点击进入项目详情

### 4.5 管理员功能
- [ ] 查看所有用户创建的项目
- [ ] 查看任意项目的材料详情

## 5. 页面结构

```
/                   -> 登录页
/dashboard          -> 仪表盘（新建项目 / 历史项目）
/projects/new       -> 新建项目表单
/projects/[id]      -> 项目详情（材料上传）
/projects           -> 历史项目列表（普通用户）
/admin              -> 管理后台（管理员）
```
